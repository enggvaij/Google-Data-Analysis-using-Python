<h2><a class="anchor" id-="business-problem">Business-Problem</h2>

This project gives the which keyword is searched how many times , in which time period and in which region of the world. 

Using this project , one can create a keyword search variable  and assign the search string e.g. "Python Programming", then using python coding access the keyword based data from google website using python library and then generate the SEO based data visualisation chart over the time, country wise.

<h2><a class="anchor" id-="dataset">Dataset</h2>

Access the data from the google using pytrends python library. Pytrends is an unofficial Python library that allows east access to Google trends data through Google's internal API. Pytrends acts as a smart wrapper around Google's web interface . enabling data analyst and developers to extract trend data for various keywords, region, and timeframes.
